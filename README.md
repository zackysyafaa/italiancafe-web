# ☕ Italian Cafe - Digital Menu & Landing Page

Proyek ini adalah contoh **Landing Page & Digital Menu** modern untuk UMKM (Cafe/Resto) yang dibuat menggunakan **HTML & Tailwind CSS**.

## ✨ Fitur Utama
- **Responsive Design:** Tampilan estetik di HP maupun PC.
- **Digital Menu:** Daftar menu yang rapi, tanpa perlu download PDF/Google Drive.
- **Direct WA Integration:** Pengunjung bisa langsung pesan via WhatsApp.
- **Fast Loading:** Ringan dan cepat diakses.

## 🛠️ Tech Stack
- HTML5
- Tailwind CSS (via CDN)
- GitHub Pages for Hosting
